Maven, JUnit and JaCoCo project example.

## How to run and test on this project?
```
Steps (!):
$ git clone https://github.com/mkyong/maven-examples.git
$ cd maven-code-coverage
$ mvn clean test

# View report at 'target/site/jacoco/index.html'
```
